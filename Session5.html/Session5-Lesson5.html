5. Groups of selectors

A comma-separated list of selectors represents the union of all elements selected by each selector in the list. (A comma is U+002C.) For example, in CSS when several selectors share the same declarations, they may be grouped into a comma-separated list. White space may appear before and after the comma.

CSS example:

In this example, we condense three rules with identical declarations into one.

h1 { font-family: sans-serif }
h2 { font-family: sans-serif }
h3 { font-family: sans-serif }
    
is equivalant to:
    
h1, h2, h3 { font-family: sans-serif}
